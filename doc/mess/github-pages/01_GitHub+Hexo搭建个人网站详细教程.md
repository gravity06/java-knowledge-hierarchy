# 一、hexo环境搭建

具体过程请参见：[hexo官方文档](https://hexo.io/zh-cn/docs/)

## 1.安装hexo

```bash
npm install -g hexo-cli
```



## 2.建站

```bash
hexo init blog
cd blog
npm install
```



新建完成后，会生成如下目录：

```bash
.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes
```
